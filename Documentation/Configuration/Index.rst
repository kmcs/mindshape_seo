Configuration Reference
=======================
The main configuration can be done in the SEO settings module. To use the page and image sitemap you have to setup a redirect in your robots.txt.

robots.txt
----------
Add the following entries to your robots.txt file. Don't forget to replace the URL with your domain!

::

    Sitemap: http://yourdomain.url/sitemap.xml
    Sitemap: http://yourdomain.url/sitemap-index.xml
    Sitemap: http://yourdomain.url/sitemap-image.xml
