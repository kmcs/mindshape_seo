Konfigurationsreferenz
======================
Alle primären Einstellungen können im "SEO Einstellungen" Modul vorgenommen werden. Um die Seiten und Bilder Sitemap zu verwenden ist es nötig Einträge in die robots.txt hinzuzufügen.

robots.txt
----------
Füge folgende Einträge zu deiner robots.txt Datei hinzu. Vergiss nicht deine eigene Domain in die URL einzusetzen!

::

    Sitemap: http://yourdomain.url/sitemap.xml
    Sitemap: http://yourdomain.url/sitemap-index.xml
    Sitemap: http://yourdomain.url/sitemap-image.xml
