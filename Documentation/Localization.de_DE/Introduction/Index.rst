Einleitung
==========

Was tut die Extension?
----------------------

Diese Extension bietet Ihnen alle möglichen Funktionen
zum Optimieren der SEO Ihrer Website.

Vorzüge von mindshape_seo
-------------------------

+ SEO Konfigurationen pro Domain

+ Neue und umsortierte Felder in den Seiteneigenschaften

+ Sitemap.xml Generator (inkl. unter Sitemap Option für einelne Seitenbäume)

+ Google Vorschau für Seiten

+ Facebook Metadaten

+ Metadaten Validierung
